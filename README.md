📊 PhonePe Transaction Analysis Dashboard

An interactive Power BI dashboard built to analyze digital payment transactions, user activity, and service performance. This project focuses on turning raw transaction data into meaningful insights using Power BI, Power Query, and DAX.

📌 About the Project
Digital payment platforms generate a huge amount of transactional data every day. The objective of this project was to explore that data, identify trends, and build an easy-to-understand dashboard that helps users quickly understand transaction performance and customer behavior.

The dashboard provides an overview of transaction volume, total transaction value, user engagement, payment success rate, service-wise performance, and monthly trends

 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel (Dataset)

 📈 Dashboard Highlights
- Total Transactions KPI
- Total Transaction Value
- Unique Users
- Payment Success Rate
- Monthly Transaction Trend
- Age Segment Contribution
- Service-wise Transaction Analysis
- Top 5 Users by Transaction Value
- Weekday vs Weekend Usage
- Interactive Filters for Month and Payment Status
- Custom Insight Panel

📊 Key Insights
- More than 257K transactions were analyzed.
- Total transaction value exceeded ₹3.38 Billion.
- Payment success rate remained above 95%.
- Loan services generated the highest transaction value.
- Weekday transactions were significantly higher than weekend transactions.
- July recorded one of the highest transaction volumes during the year.
